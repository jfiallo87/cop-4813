About COP4813

Course Description

This course is taught at Daytona State College as part of the Information System program. The course is designed to give students a head start into the field of web server programming. This is a large and dynamic field with lots of tools. The course requires students to create real web applications, that will be available on a real web site that is freely accessible.

Course Outcomes

  1. Understand and use all the capabilities of modern web browsers
  2. Ability to use XHTML capabilities to design and implement a web site
  3. Use XML, XSL, CSS, and correct techniques to create properly structured and well designed web sites
  4. Ability to use client side scripting languages (DHTML, Javascript) to add interactivity to web sites
  5. Ability to deploy web based designs in a server environment
